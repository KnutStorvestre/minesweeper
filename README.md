# How to play
* Left click to make cell visible
* Right click to flag
* You win when all mines are flagged and all other cells are visible

# Difficulty
* Beginner: 10 mines on a 9x9 grid
* Intermediate: 40 mines on a 16x16 grid
* Expert: 99 mines on a 16x30 grid
